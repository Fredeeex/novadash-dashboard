# 📊 NovaDash — Analytics Dashboard (Glass UI)

A professional, client-ready **analytics dashboard** built with **HTML, CSS, and Vanilla JavaScript**.

Includes glassmorphism UI, dark/light theme toggle, responsive sidebar, KPI cards, dynamic charts (CSS), searchable + sortable table, modal details, add transaction flow, and CSV export — **no libraries**.

## ✨ Features

- Modern Glassmorphism UI
- Dark / Light theme toggle (saved in LocalStorage)
- Responsive layout (desktop + mobile)
- Collapsible sidebar + mobile menu
- KPI cards with period deltas
- Revenue trend bars chart (CSS) from real JS data
- Status donut chart (CSS) with live percentages
- Transactions table:
  - Search
  - Filter by status
  - Sort by date / amount
- Transaction details modal
  - Mark as paid
  - Delete transaction
- Add transaction modal with validation
- CSV export of filtered data
- Scroll progress indicator

---

## 🛠 Tech Stack

- HTML5
- CSS3 (Grid / Flex + Glass UI + CSS charts)
- Vanilla JavaScript (ES6)
